# 技術スタック

## Frontend

- **Framework**: Next.js (App Router)
- **State**: Redux Toolkit
- **UI**: shadcn/ui + Tailwind CSS
- **Validation**: Zod
- **Forms**: react-hook-form

## Backend

- **Runtime**: Node.js
- **Database**: PostgreSQL + Prisma ORM
- **API**: Next.js API Routes

## Development

- **Package Manager**: npm
- **Language**: TypeScript
- **Testing**: Jest
- **Linting**: ESLint + Prettier
